## Get Debian Ready

Install basic dependencies

```sh
apt update && \
apt upgrade -y \
apt-transport-https \
ca-certificates \
curl \
gnupg-agent \
software-properties-common \
git \
build-essential \
python3-dev \
python3-venv \
python3-pip \
x264 \
ffmpeg
```
